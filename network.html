    Internet Four-Layer Software

  • application layer
  • transport layer
  • network layer
  • link layer
    •  
  • Consists of those software units such as clients and servers that use Internet Communication to carry out their tasks. This where a message typically originates.
  • Its task is to accept messages from the application layer and to ensure that the messages are properly formatted for transmission over the Internet.
  • Its task is to decide in which direction a packet, received from the transport layer, should be sent at each step along the packet's path through the Internet.
  • Its task is to actually transmit the packet up to its netowrk layer where the packet's final desitination is compared to the network layer's forwarding table to determine the direction of the packet's next step.

  • malware
  • virus
  • worm
  • trojan horse
  • spyware
    •  
  • phishing
  • DDos attack
    		•
  • The collective name of malicious software used to attack networks.
  • Software that infects a computer by inserting itself into programs that already reside in the machine.
  • An autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers.
  • A program that enteres a computer system disguised as a desirable program, such as a game or useful utility package, that is willingly imported by the victim.
